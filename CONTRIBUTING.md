# Contributing to dotnet-feature-flags

We welcome contributions!

## How to Contribute

1. Fork and clone the repository.
2. Create a feature branch: `git checkout -b feature/your-name`.
3. Develop your changes (ensure they build and pass tests).
4. Follow existing conventions (C# 13, XML documentation required).
5. Open a Pull Request with a clear description.

## Requirements

- .NET 10.0 SDK
- Follow project code style.
- Include unit tests for new functionality.

## Reporting Issues

Open an issue on GitHub with:
- Clear, descriptive title.
- Steps to reproduce (for bugs).
- Relevant system information.
