# v2.0 Basic Usage Example

This example demonstrates the new v2.0 features of the Feature Flags library, including:
- A/B testing with metrics collection
- Enhanced user context with device and session information
- Experimentation features
- Gradual rollout tracking

## Prerequisites

- .NET 10 SDK

## Running the Example

```bash
dotnet run
```

## Features Demonstrated

1. Simple feature flag evaluation
2. A/B test with metrics tracking
3. Conversion tracking for experiments
4. Experiment metrics retrieval
5. Gradual rollout with progress tracking

The example shows how to use the new v2.0 experimentation features including metrics collection for A/B tests and gradual rollouts.
